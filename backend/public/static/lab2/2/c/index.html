Introduction to DevPortal

DevPortal provides the infrastructure to build, deploy, and scale modern web applications globally. Whether you are building a static site or a complex microservices architecture, we have the tools you need.

Installation

The easiest way to interact with DevPortal is using our official Command Line Interface. It requires Node.js 18 or later.

1
$ npm install -g devportal-cli

Authentication

All API requests must be authenticated using a Bearer token. You can generate a Personal Access Token in your account settings dashboard under the Security tab.

12345
# Authenticate the CLI interactively
$ devportal login
# Or use it via standard cURL
$ curl -H "Authorization: Bearer YOUR_TOKEN" https://api.devportal.com/v1/user
